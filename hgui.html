关于HGUI插件

    你可以使用HGUI在目前所有支持HGUI的应用中使用，HopWeb拥有最佳的开发HGUI的环境，建议你使用HopWeb开发HGUI，在下面，你可以学习到开发HGUI的一些方式。

下载HopWeb

    HopWeb是一款出色的开发个人网站的工具，目前你可以在本站(hopweb.venter.club)中下载。

    HopWeb目前拥有最新的HGUI环境库，你可以尽情用它开发适用于大部分情况的插件，不过，HopWeb无内置第三方方法，也就是说，在你开发一些拥有第三方方法的插件时，可能会出错。

1.基础交互类(H)

                                                                                                                                                                             
方法名参数作用
printString messgae 显示的消息向输出面板继续增加值为message的字符串。
printlnString messgae 显示的消息向输出面板继续增加值为message的字符串并进行换行。
toastString messgae 显示的消息,int time 持续时间，0或1向主界面发送一条只为message的吐司。
showOutputboolean p1确定是否显示输入对话框。
alertString title 对话框的标题,String message 对话框的信息,String messageColor 对话框的信息颜色（HEX值）,boolean cancelable 按返回键或点击对话框空白处能否关闭对话框,String OKText 确定按钮显示的文字向主界面显示一个对话框。
addViewView v向主界面最顶端的LinearLayout增加一个View。
removeViewView v向主界面最顶端的LinearLayout移除一个View。
removeAllView无参数向主界面最顶端的LinearLayout移除所有View。
newThreadString name 最外部的方法名及参数,int id新建一个ID为id的WebView执行名为name的方法。
setErrorString e 错误信息发送一条错误信息。

3.适用于Minecraft Builder的第三方方法(builder)

             
方法名参数作用
IsOpenFile 无参数 获取当前是否打开文件的布尔值。
IsHasMobs 无参数 获取当前是否拥有生物的布尔值。
getMobsIndex 无参数 获取当前生物的序列。
getBonesIndex 无参数 获取当前部位的序列。
getBonesName 无参数 获取当前部位的名称。
addCubes float[] cube 方块信息数组,String tag 颜色标签值(HEX值),String describe(注释值),int mobIndex 指定生物序列,int boneIndex 指定部位序列,String boneName 指定部位名称 增加方块，并返回是否成功的布尔值。
addBone String json JSON文本 增加部位，并返回是否成功的布尔值。
removeBone int boneIndex 移除部位，并返回是否成功的布尔值。
addMob String mobCode 生物代号,String mobName 生物名称,String json。JSON文本,int[] textureSize 贴图大小数组(width,height) 增加生物，并返回是否成功的布尔值。
removeMob int mobIndex 生物序列 移除生物，并返回是否成功的布尔值。
